# YERIM LOG

예림의 GitHub Pages 블로그/포트폴리오 리포지토리입니다.

- Theme: Jekyll + Minimal Mistakes (`contrast` skin)
- Main content: 프로젝트 기록, 공부 노트, 메모

## URL

- 사용자 페이지 리포 이름이 `ye-rim-oh.github.io`이면: `https://ye-rim-oh.github.io/`
- 현재 리포 이름이 `yerim.github.io`이면: `https://ye-rim-oh.github.io/yerim.github.io/`

## Local preview (optional)

```bash
bundle install
bundle exec jekyll serve
```

Open `http://127.0.0.1:4000`.
